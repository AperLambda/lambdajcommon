## Adding 位Jcommon to your project

It's very simple!

### Gradle

In your repositories declaration add:
```groovy
maven { url "https://lambdaurora.github.io/repositories" }
```

And in your dependencies add:
```groovy
compile 'com.aperlambda:lambdajcommon:1.1.0'
```
